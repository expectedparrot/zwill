# Agent Instructions

For end-to-end survey ingestion, digital-twin validation, one-shot marginal
baselines, generated reports, and gated readouts, use the bundled CLI guide as
the source of truth:

```bash
zwill guide
zwill next
```

Run `zwill next` after each completed stage and follow the returned
`next_steps`. The guide contains the ingestion, codebook, approval, remote
inference, leakage, sample-size, artifact, final-report, and interpretation
rules. If the guide and this file diverge, update the guide and keep this file as
only the bootstrap pointer.
